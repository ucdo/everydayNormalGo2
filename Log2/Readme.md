## 实现一个日志库
1. 只记录关键字，具体的根据文档自己实现

### 日志库需求
1. 支持文件写入日志
2. 支持往终端打印日志

### 需求分解
1. DEBUG 测试的日志 
2. TRACE 链路日志
3. INFO 信息日志
4. WARNING 警告日志
5. ERROR 错误日志
6. FATAL 严重错误

7. 要包含时间，日志级别，调用的文件，调用的函数，信息
8. 提示： 参考 fmt.Fprintf
9. 只有大于某一个日志级别才记录日志
10. 提示：某一行直接调用runtime包里面的内容
11. path.Base() runtime.Caller()

12. 实现文件大于 10M （调整成可以配置的） 即切分

13. 这里应该需要加锁吧，切换句柄的时候
